# Bubble Tag — Apresentação do Projeto

Apresentação web do projeto **Bubble Tag**, um jogo de apanha inclusivo para a plataforma Roblox. Esta página explica o conceito, papéis de jogo, mecânicas, design e ferramentas utilizadas, num formato narrativo e visual.

## Visão geral
O projeto destaca uma abordagem inclusiva ao jogo do apanha: jogadores capturados ficam presos numa bolha e podem ser libertados pelos colegas, mantendo todos na partida até ao fim. A apresentação foca-se em:
- Conceito e objetivos do design
- Papéis (Runner e Chaser)
- Mecânica central da bolha
- Game loop, pontuação e economia
- Design visual e UX
- Ferramentas e processo de desenvolvimento

## Como visualizar
1. Faça download/clonagem do repositório.
2. Abra o ficheiro `index.html` no navegador.

## Estrutura do repositório
- `index.html` — página principal da apresentação
